SScript 1.0-gitrc

A high-performance hybrid language mixing Python's readability with C++'s native speed.

>>> var url = "cdn.jsdelivr.net/..."
>>> Net.fetch(url, "lib.js")
[BUILD] Binary 'main' created.

Installation

Download sscript.py and run the following in your terminal:

chmod +x sscript.py
sudo cp sscript.py /usr/local/bin/sscript
